Utility
==========

> Utility - utility package - misc helpers

## Install

> Using Composer and Packagist

in composer.json file add:

```json
{
    "require": {
        "g4/utility": "*"
    }
}
```

## Usage

## Development

### Install dependencies

    $ make install

### Run tests

    $ make test
